# 🎮 GameForge AI - AI-Powered Game Development Platform

An advanced AI-powered platform for game developers featuring intelligent tools for scripting, 3D modeling, building, debugging, visual effects, and learning.

## ⚡ Features

### 🤖 AI Tools
- **📝 Script Generator** - Generate Lua scripts for Roblox and code for Unity, Godot, and Unreal
- **🎨 Model Assistant** - Create 3D props, characters, and textures from text prompts
- **🏗️ Building Tool** - Generate maps, terrain, and structures instantly
- **🐛 Code Debugger** - AI-powered debugging with error analysis and suggestions
- **✨ VFX Creator** - Create particle effects, animations, and UI elements
- **🤖 AI Mentor** - Interactive tutorials, challenges, and AI mentorship

### 📊 Dashboard
- Real-time project statistics
- Daily credit limit tracking (100 credits/day)
- Quick access to all tools
- Project management hub
- Learning center with courses
- Daily challenges with XP rewards

### 🎨 Design
- Dark futuristic neon theme
- Animated backgrounds with twinkling stars
- Smooth transitions and hover effects
- Fully responsive (mobile, tablet, desktop)
- Color-coded UI with blue, purple, cyan, green, and pink accents

## 🚀 How to Use

1. Open the platform in your browser
2. Click on any AI tool to open its interactive panel
3. Follow the tool's instructions to generate content
4. Monitor your daily credit usage in the top navigation bar
5. Explore projects, learning materials, and challenges

## 📁 Files

- **index.html** - Main dashboard and UI structure
- **styles.css** - Dark futuristic styling with neon theme
